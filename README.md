# Hypr Setup

A minimalist, modern Hyprland configuration setup.

## Installation

### Quick Install

Install the complete system with a single command:

```bash
curl -sSL https://raw.githubusercontent.com/benitez96/hypr-setup/main/boot.sh | bash
```

### Manual Install

1. Clone the repository:
```bash
git clone https://github.com/benitez96/hypr-setup.git ~/.local/share/hypr
cd ~/.local/share/hypr
```

2. Run the installation:
```bash
bash install.sh
```
